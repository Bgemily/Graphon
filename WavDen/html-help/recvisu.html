recvisu

 

VisuShrink Denoising Procedure

 

DESCRIPTION

 
Denoising a signal using the VisuShrink procedure of Donoho & Johnstone (1994).

 

USAGE

 

      f = recvisu(signal, type, h)

 

REQUIRED ARGUMENTS

 

signal

 

1-d noisy signal, length(signal)= 2^J

type

 

‘H’ for hard, ‘S’ for soft

h

 

Quadature mirror filter for wavelet transform

 

OPTIONAL ARGUMENTS

 

h    Optional, Default=Symmlet 8

 

VALUE

 

f          Estimate, obtained by applying the VisuShrink procedure on the wavelet coefficients.

 

REFERENCES

 

See WaveLab

 

EXAMPLES

 

h = MakeONFilter('Symmlet', 8);

reconstruct = recvisu(signal,'H', h);
