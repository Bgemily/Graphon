recmixed

 

Mixed-Effects Denoising Procedure

 

DESCRIPTION

 
Denoising a signal using a mixed-effects model. It calls the gcvme procedure.

 

USAGE

 

      f = recmixed(signal, h)

 

REQUIRED ARGUMENTS

 

signal

 

1-d noisy signal, length(signal)= 2^J

h

 

Quadature mirror filter for wavelet transform

 

 

 

OPTIONAL ARGUMENTS

 

h          Optional,  Default  =  Symmlet 8

 

VALUE

 

f          Estimate, obtained by applying Bayesian wavelet shrinkage for nonparametric

             mixed-effects models.

 

SIDE EFFECTS

 

           Uses WaveLab and the functions gcvme and fmin in  the optimisation

           toolbox  of Matlab. The range of the smoothing parameter is [0.01, 5] in

          

                               fmin('gcvme', 0.01,5)

 

           which make it different from Table 1 in Huang & Lu (2000).

 

REFERENCES

 

Huang, S.Y. & Lu, H. H.-S. (2000).  Bayesian wavelet shrinkage for nonparametric

mixed-effects models. Statistica Sinica, 10,  1021-1040.
