postblockmed

 

Bayesian Block Denoising Procedure

 

DESCRIPTION

 
Denoising empirical wavelet coefficients by grouping them into disjoint blocks of length l, and estimates them by

using their blocking posterior median. It is needed by the recblockmed and rechybblockmed procedures.

 

USAGE

 

f = postblockmed(method, wcoef,l, sigma, thet0)

 

REQUIRED ARGUMENTS

 

method

 

'Augment' or 'Truncate'

wcoef

 

Empirical wavelet coefficients at a given resolution

l

 

Length of the blocks

sigma

 

Level of the noise

thet0

 

Parameter vector (initial values for maximization)

  

OPTIONAL ARGUMENTS

 

VALUE

 

f          Vector of denoised wavelet coefficients.

 

BACKGROUND


This is an (individual) thresholding estimator where each empirical wavelet coefficient is thresholded utilizing

information about neighboring coefficients within a block. It is called the PostBlockMed estimator. At

each resolution level j (j=j0,...,J-1), the wavelet coefficients are grouped into nonoverlapping

blocks of length l[j]. In each case, the first few empirical wavelet coefficients might be re-used to fill

the last block (which is called the Augmented case) or the last few remaining empirical wavelet coefficients

might not be used in the posterior-based inference (which is called the Truncated case), should l[j] not

divide 2^j exactly.

 

REFERENCES

 

            Abramovich, F., Besbeas, P. & Sapatinas, T. (2000). Empirical Bayes approach to block wavelet function

            estimation. Technical Report, Department of Mathematics and Statistics, University of Cyprus, Cyprus.
