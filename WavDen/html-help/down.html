Download Gaussian Wavelet Denoising  Library

Note: These scripts are available freely for non-commercial use only.

Gaussian Wavelet Denoising (GaussianWaveDen)  library is a collection of  MATLAB™ scripts allowing one to denoise (one-dimensional) signals contaminated with additive Gaussian noise, using a large variety of wavelet-based statistical estimation techniques. The scripts rely upon the WAVELAB package developed at Stanford by Buckheit, Chen, Donoho, Johnstone & Scargle (1995). The entire MATLAB source code, in compressed zip form, is available for download from:

http://www-lmc.imag.fr/SMS/software/GaussianWaveDen/wavden.zip

The current maintainers (Anestis Antoniadis, Jeremie Bigot & Theofanis Sapatinas) of  GaussianWaveDen library have an accompagnying paper at:

http://www-lmc.imag.fr/SMS/software/GaussianWaveDen/wavden.pdf.zip

As distributed, the scripts are set up to be used out of the box by users of  MATLAB distributions, with installation consisting of placing the required files in an appropriate directory in a location searched by MATLAB via its path.
