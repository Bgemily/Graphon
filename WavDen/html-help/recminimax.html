recminimax

 

Minimax Denoising Procedure

 

DESCRIPTION

 
Denoising a signal using the WaveThresh procedure of  WaveLab with minimax thresholds.

 

USAGE

 

       f = recminimax(signal, type, h)

 

REQUIRED ARGUMENTS

 

signal

 

1-d Noisy signal, length(signal)= 2^J

type

 

 'H' for hard; 'S' for Soft

h

 

Quadature mirror filter for wavelet transform

 

OPTIONAL ARGUMENTS

 

h    Optional, Default=Symmlet 8

 

VALUE

 

f          Estimate, obtained by applying the WaveThresh procedure of  WaveLab with minimax thresholds.

 

REFERENCES

 

See  WaveLab

   

EXAMPLES

 

h = MakeONFilter('Symmlet',8);

reconstruct = recminimax(signal, 'H', h);
