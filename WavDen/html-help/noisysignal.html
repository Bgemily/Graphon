noisysignal

 

Creating a Noisy Signal

 

DESCRIPTION

 
Generatea a noisy signal from a specified function.

USAGE

 

f = noisysignal(name, n, rsnr)

 

REQUIRED ARGUMENTS

 

name

 

A string: 'Step','Wave','Blip','Blocks','Bumps','HeaviSine',

 

 

'Doppler','Angles','Parabolas','Time Shifted Sine','Spikes','Corner'

n

 

Desired signal length

rsnr

 

Signal to noise ratio

OPTIONAL ARGUMENTS 

VALUE

 

f          1-d noisy signal

 

BACKGROUND

 

This procedure uses the MakeSignalNewb MATLAB procedure. The noise added is Gaussian.

 

REFERENCES

 

            Buckheit, J.B. & Donoho, D.L. (1995).WaveLab and Reproducible Research.
            In Wavelets and Statistics, Antoniadis, A.& Oppenheim, G. (Eds.), Lect. Notes
            Statist., 103, pp. 55--81, New York: Springer-Verlag.

   

SEE ALSO

 

MakeSignalNewb

 

EXAMPLES

 

names={'Step' 'Wave' 'Blip' 'Blocks' 'Bumps' 'HeaviSine' ...
'Doppler' 'Angles' 'Parabolas' 'Time Shifted Sine' ...
'Spikes' 'Corner'};

n=128; rsnr=3;

signal = noisysignal(names(4), n, rsnr);
