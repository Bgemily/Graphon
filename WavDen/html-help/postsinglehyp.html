postsinglehyp

 

Bayesian Denoising Procedure

 

DESCRIPTION

 

Estimate the wavelet coefficients using a hypothesis testing method needed by the recsinglehyp procedure .

 

USAGE

 

f = postsinglehyp(wcoef, sigma, thet0)

 

REQUIRED ARGUMENTS

 

wcoef

 

Empirical wavelet coefficients at a given resolution level

thet0

 

2 component vector containing the initial values for the EM algorithm

sigma

 

Noise level

 

OPTIONAL ARGUMENTS

 

VALUE

 

f           Denoised wavelet coefficients

  

BACKGROUND

 

              This is a method similar in spirit to the multiple hypotheses testing procedures  implemented in recfdr.

              It is a Bayesian method, considered by Vidakovic (1998),  for obtaining a bonafide wavelet thresholding

              estimator. See also Abramovich & Sapatinas (1999).
