recfdr

 

Multiple Comparison Denoising Procedure

 

DESCRIPTION

 

Denoising a signal using the false discovery rate method.

 

USAGE

 

f = recfdr(signal, type, h)

 

REQUIRED ARGUMENTS

 

signal

 

1-d noisy signal, length(signal)=2^J

type

 

 'S' for soft, 'H' for hard

h

 

Quadratic mirror filter

   

OPTIONAL ARGUMENTS

 

h         Optional,  Default=Symmlet 8

 

VALUE

 

f           Estimate, using the false discovery rate method on the wavelet coefficients.

 

BACKGROUND

 

The idea of wavelet thresholding can be viewed as a multiple hypotheses testing. Classical approaches to multiple

hypotheses testing in a wavelet setting face serious problems because of the large numbers of hypotheses being

tested simultaneously. In other words, if the error is controlled at an individual level, the chance of keeping erroneosly

a coefficient is extremely high; if the simultaneous error is controlled, the chance of keeping a coefficient is extremely low.

Abramovich & Benjamini (1995,1996) proposed a way to control such dissipation of power based on the false discovery

rate.
